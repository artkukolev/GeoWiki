# 🌍 GeoWiki - Премиум Образовательный Сайт

**ВАУ-эффект гарантирован!** 🚀

Интерактивная образовательная платформа по географии и условным знакам, созданная по принципам топ-стартапов. Современный дизайн уровня Stripe / Notion / Apple с glassmorphism, neumorphism и премиум UX.

## ✨ Что делает этот сайт особенным

### 🎨 Премиум Дизайн
- **Glassmorphism & Neumorphism** - Прозрачные элементы с размытием
- **Градиенты уровня Dribbble** - Плавные цветовые переходы
- **Мягкие тени** - Реалистичные эффекты глубины
- **Большие отступы** - Air spacing для комфорта
- **Чёткая типографика** - Современные шрифты Inter

### ⚡ Премиум Функционал
- **Custom Cursor** - Интерактивный курсор с анимацией
- **Preloader** - Стильный загрузчик при старте
- **Анимированный фон** - Градиент с плавным сдвигом
- **Scroll Reveal** - Плавное появление элементов
- **Мини-игра** - "Угадай знак" для закрепления знаний
- **Hover эффекты** - Микроанимации на всех элементах

### 🎯 UX Мастерства
- **Интуитивная навигация** - Всё понятно без объяснений
- **Визуальное обучение** - Минимум текста, максимум интерактива
- **Цветовая кодировка уровней** - Каждый уровень имеет свой цвет
- **Прогресс визуализация** - Круговой прогресс-бар
- **Достижения** - Система наград за успехи

## 🚀 Запуск и Публикация

### Локальный Запуск
```bash
cd GeoWiki
python3 -m http.server 3000
# Откройте http://localhost:3000 в браузере
```

### Публикация на GitHub Pages
1. Загрузите проект на GitHub
2. В настройках репозитория включите GitHub Pages
3. Выберите ветку main и папку root
4. Сайт будет доступен по адресу: `https://username.github.io/repository-name`

### Публикация на Другой Хостинг
Проект полностью статический, поэтому подходит для:
- Netlify
- Vercel
- Firebase Hosting
- AWS S3 + CloudFront
- Любого статического хостинга
