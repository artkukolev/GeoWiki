# 🌍 GeoWiki 2.0 - Премиум Образовательная Платформа по Географии

## 📌 Обзор Проекта

**GeoWiki** — интерактивная географическая платформа нового поколения, созданная для изучения стран, карт и навигации по миру. Это продукт уровня коммерческого образовательного сервиса.

### ✨ Ключевые Особенности

- 🗺️ **Интерактивная Карта Мира** с 50+ странами
- 🎓 **5 Уровней Обучения** от базового до профессионального
- 📚 **Энциклопедия Стран** с подробной информацией
- 🎮 **Мини-Тесты & Игры** для закрепления знаний
- 👤 **Профиль Пользователя** с достижениями и прогрессом
- 🎨 **Премиум Дизайн** с современным UI/UX
- 📱 **Мобильная Адаптация** (responsive design)

---

## 🎯 Структура Сайта

```
GeoWiki/
├── index.html              # Главная страница
├── map.html               # Интерактивная карта (Leaflet.js)
├── encyclopedia.html      # Страница информации о странах
├── profile.html           # Профиль пользователя
├── symbols/
│   └── index.html         # Условные знаки (оригинальный контент)
├── levels/
│   ├── beginner.html      # Уровень 1: Начальный
│   ├── confident.html     # Уровень 2: Уверенный
│   ├── expert.html        # Уровень 3: Эксперт
│   ├── professional.html  # Уровень 4: Профессиональный
│   └── geographer.html    # Уровень 5: Географ
├── css/
│   ├── design-system.css  # Дизайн-система (переменные, цвета, типографика)
│   └── style.css          # Стили страниц
└── js/
    ├── app.js             #主 контроллер приложения
    ├── main.js            # Основные анимации и интерактивность
    ├── progress.js        # Отслеживание прогресса
    ├── modal.js           # Система модальных окон
    ├── quiz.js            # Логика тестов
    ├── mini-game.js       # Мини-игры
    ├── filter.js          # Фильтрация контента
    └── countries-data.js  # БД со странами
```

## 🎨 Дизайн-Система

### Цветовая Палитра (PREMIUM DARK THEME)
- **Primary Blue**: `#4DA3FF`
- **Secondary Green**: `#7CFFB2`
- **Accent Red**: `#FF6B9D`
- **Dark Background**: `#0B1220`
- **Text Primary**: `#FFFFFF`

### Типографика
- **Шрифт**: Inter (700-900 weight для заголовков)
- **Иерархия**: Крупные, контрастные размеры
- **Spacing**: Большие отступы для "воздухопроницаемости"

### UI Компоненты
- ✅ Glassmorphism карточки (blur + прозрачность)
- ✅ Градиентные кнопки с анимацией
- ✅ Мягкие тени (box-shadow)
- ✅ Плавные переходы (transitions)

## 🗺️ Интерактивная Карта

### Технология
- **Leaflet.js** (открытая библиотека для карт)
- **OpenStreetMap** (бесплатная база карт)
- **Кастомные маркеры** с флагами стран

### Функциональность
1. Карта мира с 50+ странами
2. Клик по стране → модальное окно с информацией
3. Hover-эффекты для подсветки
4. Ссылка на энциклопедию для каждой страны

**URL**: `/map.html`
